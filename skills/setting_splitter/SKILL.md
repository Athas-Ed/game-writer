# 设定拆分器技能

## 功能说明
- 输入：一个包含多类设定的长文本（通常为 `.txt` 或 `.md` 的内容粘贴）
- 输出：按“人物/种族/阵营/地图/历史”等类型自动拆分后的多个 Markdown 文件，并生成对应目录结构

## 工作流程
1. **读取与分割**：将输入文本按双换行符（`\n\n`）分割为段落块。
2. **分析与识别**：对每个段落块识别标题行与关键词（如 `##/###`、`人物：`、`种族：` 等），判断条目类型并提取条目名称。
3. **分类映射**：将类型映射到目录（默认映射见下方“目录映射”）。
4. **生成文件名**：使用条目名称清理非法字符生成文件名（追加 `.md`）。
5. **格式化内容**：确保条目输出为规范 Markdown（如 `## 条目名` + 列表项 `- **键**：值`）。
6. **保存文件**：自动创建目录并写入文件；同时输出本次写入的文件清单。

## 目录映射（默认）
- `人物/角色` → `data/角色设定/`
- `种族` → `data/背景设定/种族/`
- `阵营` → `data/背景设定/阵营/`
- `地图` → `data/背景设定/地图/`
- `历史` → `data/背景设定/历史/`
- 其他/无法识别 → `data/背景设定/未分类/`

## 使用示例
输入片段：

## 人物：艾莉亚·星影
- 种族：高等精灵
- 阵营：银月联盟
- 职业：法师
- 描述：一位年轻的高等精灵法师，擅长星辰魔法，性格冷静。

## 种族：高等精灵
- 特征：尖耳、长寿、魔法亲和
- 分布：银光森林
- 历史：上古魔法文明的继承者。

输出文件（示例）：
- `data/角色设定/艾莉亚·星影.md`
- `data/背景设定/种族/高等精灵.md`
